# Odoo Timesheet Timer Extension

VS Code and Cursor extension for managing Odoo 16 timesheet timers.

## Features

- Login to Odoo 16 instance
- Start/Stop timesheet timer
- View current timer status and duration
- List recent timesheet entries
- Secure credential storage

## Usage

1. Configure Odoo server URL in settings
2. Use `Odoo: Login` command to authenticate
3. Use `Odoo: Start Timer` to start a timesheet entry
4. Use `Odoo: Stop Timer` to stop the current timer
5. Use `Odoo: Show Timer Panel` for full interface

## Development

```bash
npm install
npm run compile
npm run package
```
